    MongoosePush v2.0.0 API Reference

    Modules

    MongoosePush

    MongoosePush is simple (seriously) service providing ability to send push notification to FCM (Firebase Cloud Messaging) and/or APNS (Apple Push Notification Service). What makes it cool is not only simplicity but also support for newest and fastest HTTP/2 based APIs for both services.

    MongoosePush.API

    Module for handling internal responses to V1 HTTP2 codes

    MongoosePush.API.V2.ResponseEncoder

    Module for handling internal responses to V2 HTTP2 codes

    MongoosePush.API.V3.ResponseEncoder

    Module for handling internal responses to V3 HTTP2 codes

    MongoosePush.Metrics

    MongoosePush.Service

    Generic interface for push notifications services.

    MongoosePush.Service.APNS

    APNS (apple Push Notification Service) service provider implementation.

    MongoosePush.Service.APNS.ErrorHandler

    Module responsible for handling errors returned by APNS service

    MongoosePush.Service.APNS.State

    Module for storing state of apns APNS configurations, namely default topics

    MongoosePush.Service.APNS.Supervisor

    APNS module supervising Sparrow's PoolSupervisor and APNS State

    MongoosePush.Service.FCM

    FCM (Firebase Cloud Messaging) service provider implementation.

    MongoosePush.Service.FCM.ErrorHandler

    Module responsible for handling errors returned by FCM service.

    MongoosePush.Service.FCM.Pool.Supervisor

    This module is responsible for setting up Sparrow's FCM Supervisor

    MongoosePush.Service.FCM.Pools

    This module is responsible for worker pools management. It provides several utility functions that help with e.g. selecting workers for given pool of the service.
